# 物资管理开发说明

依据：`Data-Dictionary.md` §4、`API-Index.md` §3、`Permission-Design.md` §2。

## 首期目标

- 物资分类、物资档案、供应商、仓库。
- 库存余额查询。
- 入库、出库、盘点与库存流水。

## 权限要求

- 查询类接口使用 `inventory:*:read`。
- 写入类接口使用 `inventory:*:write`。
- AI 仅允许调用物资档案列表和库存查询两个只读接口。

## 第一阶段实现清单

依据：`plans/oa20260623.plan.md`。

- 医疗用品库、非医疗用品库。
- 物资类型、默认价格、最新价格、价格记录。
- 入库/出库/盘点明细。
- 入库/出库必须由 OA approved 事件触发库存变更。
- 报销记录可关联已通过的入库 OA。
- AI 仅可查询物资、库存、价格，不可写库存。
